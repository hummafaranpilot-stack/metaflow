SEO Fix Plan

7 fixes ordered by SEO impact. Each with pros & cons. Testing on KW Desktop first.

1robots.txt + sitemap.xml

Critical

The foundation of technical SEO. Without these, search engines don't know what to crawl or where your pages are.

robots.txt — tells search engines what to crawl/skip:
  User-agent: *
  Allow: /
  Disallow: /short/
  Disallow: /popup/
  Disallow: /lib/
  Disallow: /eBooks/
  Sitemap: https://ketowater.trustednutraproduct.com/sitemap.xml

sitemap.xml — lists all indexable pages with priority & update frequency
SEO Impact
Effort

Pros

  • Directly fixes the Lighthouse robots.txt FAIL
  • Enables Google Search Console sitemap submission
  • Prevents indexing of junk pages (popup, lib, eBooks)
  • Takes 2 minutes to create
  • Zero risk — purely additive

Cons

  • None — this is mandatory for any site wanting search traffic
  • Needs to be kept updated if new pages are added

2Canonical URLs

Critical

desktop.html and mobile.html are separate pages with the same content — Google sees this as duplicate content. Canonical tags tell Google which version to index.

Add to desktop.html:
  <link rel="canonical" href="https://ketowater.trustednutraproduct.com/desktop.html">

Add to mobile.html:
  <link rel="canonical" href="https://ketowater.trustednutraproduct.com/desktop.html">

Both point to desktop as the canonical version. Google indexes desktop, serves mobile users the mobile version via index.html redirect.
SEO Impact
Effort

Pros

  • Eliminates duplicate content penalty risk
  • Consolidates link equity to one URL
  • One line of HTML per page
  • Industry standard practice

Cons

  • If set wrong, could de-index a page accidentally
  • Mobile-first indexing: Google may prefer mobile as canonical instead
  • Need to decide: desktop or mobile as canonical?

3Meta Charset at Top of <head>

High

Lighthouse flags charset declaration missing or too late. The <meta charset="utf-8"> must be within the first 1024 bytes of the document.

Add as FIRST tag inside <head>:
  <meta charset="utf-8">

Must come before any other tag, including <link> and <style>.
SEO Impact
Effort

Pros

  • Fixes Best Practices FAIL immediately
  • Prevents character encoding bugs
  • One line, zero risk

Cons

  • None — purely a fix for a missing standard tag

4Fix Heading Order (h1 → h2 → h3 → h4)

High

Currently using h4 before h3, h5 after h3, h6 randomly. Proper hierarchy helps search engines understand content structure.

Current issues:
  • h4 appears before h3 in several sections
  • h5 used directly after h3 (skipping h4)
  • h6 used without h5 parent

Fix: Re-tag headings to follow sequential order.
  h1 (page title) → h2 (sections) → h3 (subsections) → h4 (details)
SEO Impact
Effort

Pros

  • Fixes both SEO and Accessibility audits
  • Improves content hierarchy for crawlers
  • Better screen reader experience
  • Google uses heading structure to understand topics

Cons

  • Changing heading tags may affect CSS styling
  • Need to audit all headings across ~6000 line pages
  • Visual testing needed to ensure no style breaks

5Open Graph + Twitter Card Tags

Medium

When someone shares your page on Facebook, WhatsApp, Telegram, Twitter, or LinkedIn — they see a blank preview. OG tags control exactly what appears.

Add to <head>:
  <meta property="og:title" content="KetoWater™ — Keto Hydration Supplement">
  <meta property="og:description" content="Science-backed keto hydration...">
  <meta property="og:image" content="https://ketowater.trustednutraproduct.com/product_images/hero-bg.webp">
  <meta property="og:url" content="https://ketowater.trustednutraproduct.com/">
  <meta property="og:type" content="product">
  <meta name="twitter:card" content="summary_large_image">
  <meta name="twitter:title" content="KetoWater™">
  <meta name="twitter:description" content="...">
  <meta name="twitter:image" content="...">
SEO Impact
Effort

Pros

  • Rich previews when shared on social media
  • Higher click-through rates from social shares
  • Professional appearance on WhatsApp/Telegram
  • Indirect SEO signal (more shares = more traffic)

Cons

  • Doesn't directly affect Google ranking
  • Need a good OG image (ideally 1200x630)
  • Must be maintained if title/description changes

6JSON-LD Structured Data (Product Schema)

Medium

Structured data helps Google understand your page is a product with a price, rating, and availability. Can trigger rich snippets in search results (stars, price, stock).

Add before </body>:
  <script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "Product",
    "name": "KetoWater",
    "description": "Science-backed keto hydration supplement...",
    "image": "https://ketowater.trustednutraproduct.com/product_images/hero-bg.webp",
    "brand": {"@type": "Brand", "name": "Trusted Nutra Products"},
    "offers": {
      "@type": "Offer", "price": "69.00", "priceCurrency": "USD",
      "availability": "https://schema.org/InStock"
    },
    "aggregateRating": {
      "@type": "AggregateRating", "ratingValue": "4.8", "reviewCount": "2147"
    }
  }
  </script>
SEO Impact
Effort

Pros

  • Rich snippets in Google (stars, price, availability)
  • Higher click-through from search results
  • Helps Google understand page content as a Product
  • Easy to validate with Google's Rich Results Test

Cons

  • Data must match what's on the page (Google verifies)
  • Fake reviews in schema can trigger Google penalty
  • Need to update price/rating if they change
  • Rich snippets not guaranteed — Google decides

7Accessibility Fixes (contrast + button names)

Lower Priority

15 elements have insufficient color contrast. 2 gallery navigation buttons lack accessible names. Accessibility indirectly affects SEO via user experience signals.

Color contrast fixes:
  • Green highlight text (#6fb81c) → darken to #4a8c10 (contrast 3.5:1 → 4.5:1)
  • Teal accent (#3bb790) → darken to #2a8a6c
  • Strikethrough price (#888) → darken to #666
  • "Excellent" badge → increase contrast
  • "FAQS" heading → increase contrast

Button names:
  • Add aria-label="Previous image" / "Next image" to gallery nav buttons
SEO Impact
Effort

Pros

  • Improves Accessibility score (85 → 95+)
  • Better for visually impaired users
  • Google considers accessibility as quality signal
  • Screen reader compatibility

Cons

  • Color changes may affect brand design aesthetic
  • Need to visually verify each color change
  • Minimal direct SEO ranking impact
